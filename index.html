HTML a CSS Ukážka s Definíciami

Tento dokument ukazuje základné HTML tagy a obsahuje vysvetlenia ku každej téme.

Zoznam HTML tém a vysvetlenia

  • HTML tagy: nadpisy, odseky, zoznamy

    <h1> až <h6>: Tieto tagy slúžia na vytváranie nadpisov rôznej veľkosti. Číslo za 'h' určuje úroveň dôležitosti, kde <h1> je najdôležitejší.

    <p>: Používa sa na označenie odseku v HTML. Každý odsek je obalený týmto tagom.

    <ul> a <li>: Tag <ul> vytvára nečíslovaný zoznam, zatiaľ čo <li> označuje jednotlivé položky v zozname.

  • Atribúty: href, src, alt

    href: Atribút používaný v tagu <a>, ktorý špecifikuje cieľ odkazu (napr. URL alebo cesta k súboru).

    src: Atribút používaný v tagu <img> na určenie zdroja obrázka (jeho URL alebo cesta k nemu).

    alt: Tento atribút poskytuje alternatívny text pre obrázok, ak sa obrázok nezobrazí.

  • HTML odkazy a obrázky

    <a>: Tento tag vytvára odkaz na inú stránku alebo súbor. Atribút href určuje cieľ odkazu.

    <img>: Tento tag slúži na vkladanie obrázkov. Vyžaduje atribút src, ktorý určuje zdroj obrázka, a atribút alt, ktorý poskytuje alternatívny text.

  • HTML formáty: nečíslovaný a číslovaný zoznam

    <ul>: Nečíslovaný zoznam, kde položky nie sú očíslované, ale zobrazené ako bodky.

    <ol>: Číslovaný zoznam, kde každá položka má svoje poradové číslo.

  • Struktúra dokumentu HTML

    <html>: Tento tag označuje začiatok a koniec HTML dokumentu.

    <head>: Tento tag obsahuje meta informácie o stránke, ako je kódovanie, názov, a odkazy na CSS súbory.

    <body>: Obsahuje všetok viditeľný obsah na stránke.

Zoznam CSS tém a vysvetlenia

  • Výber elementov: tag, trieda, id

    V CSS môžeš vyberať elementy podľa tagov (napr. p pre odseky), podľa tried (napr. .trieda) alebo podľa identifikátorov (napr. #id).

  • Farby: text a pozadie

    Farbu textu a pozadia môžeš nastaviť pomocou vlastností color (text) a background-color (pozadie) v CSS.

  • Rozloženie a prispôsobenie (margin, padding)

    Vlastnosti margin a padding riadia rozostupy okolo a vo vnútri HTML elementov.

  • Práca s fontami a veľkosťami textu

    Použi font-family na zmenu písma a font-size na nastavenie veľkosti textu.

  • Responsívny dizajn pomocou media queries

    Media queries v CSS umožňujú prispôsobiť vzhľad stránky rôznym zariadeniam, napríklad mobilom a tabletom.
